This repository contains a **machine learning system for detecting phishing websites** using structured (tabular) website features. The project applies a **feed-forward neural network** trained with **mini-batch gradient descent** to classify websites as phishing or legitimate.

The model is trained and evaluated on the **UCI Phishing Websites Dataset**, which consists of 30 handcrafted features extracted from URLs and website characteristics.
